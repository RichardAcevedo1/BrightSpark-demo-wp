🔹 BrightSpark Website (WordPress + Elementor)

This repository contains the core assets and customizations used to build the BrightSpark website, developed with WordPress and Elementor.

The project focuses on creating a clean, responsive, and performance-oriented website using a combination of visual builder tools and custom code.


🚀 Project Overview

BrightSpark is a modern website built for showcasing a business with a strong emphasis on:

Clean UI design
Responsive layout
Performance optimization
Practical user experience

This repository includes reusable components and custom implementations used during development.


🛠️ Tech Stack
WordPress
Elementor
HTML5
CSS3 (custom styling)
JavaScript (UI interactions)


📂 Project Structure

/assets
  /images        → Optimized images used in the website

/css
  custom.css     → Custom styles (floating menu, layout tweaks)

/js
  custom.js      → UI interactions and enhancements

/elementor
  sections/      → Exported Elementor sections (JSON)

/theme
  child-theme/   → WordPress child theme files
  

✨ Features Implemented

Responsive design (desktop, tablet, mobile)
Floating header menu with scroll effect
WhatsApp floating contact button
Custom contact form styling
Structured footer layout
UI improvements using custom CSS


⚙️ Plugins & Optimization

The original project includes integration with:

Performance optimization tools

📸 Elementor Components

This repo includes exported Elementor sections that can be imported into other WordPress projects.

To use them:
Go to Elementor → Templates
Import the JSON files from /elementor/sections/
Customize content as needed


🎨 Customization

Custom styles and scripts are separated for easy maintenance:

Edit /css/custom.css for layout and design
Edit /js/custom.js for interactions

Ensuring consistent layout and usability across devices.
